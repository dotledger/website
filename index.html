Dot Ledger

Free and Open Source personal finance management.

About

Warning! Dot Ledger is not yet stable and may undergo major changes before the first release.

Dot Ledger is a web-based personal finance management tool.

The aim of this project is to create a stable, FOSS alternative to Xero Personal which was shut down in November 2014.

Features

  • Account dashboard
  • REST API
  • Spending goals
  • OFX import
  • Income and expenses report
  • Transaction categories
  • Sorting rules
  • Account groups
  • Planned payments
  • Transaction tags
  • Transaction search
  • Balance projection

Roadmap

  • Email reminders
  • More reports
  • Split transactions
  • Support other import formats (QIF etc)
  • Saved Searches
  • Transaction autocomplete
  • Track assets and liabilities
  • CSV export
  • API documentation
  • API clients
  • Hosted SaaS version product
